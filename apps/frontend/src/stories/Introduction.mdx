# Trainer App — Component Library

This is the living documentation for every UI primitive in the Trainer App.

Components are built against these principles:

- **`forwardRef` throughout** — every form control forwards its ref so React Hook Form
  and test utilities can reach the underlying DOM element
- **`useId` for label associations** — every Input, TextArea, and Select generates a
  stable, SSR-safe ID pair linking `<label>` to `<input>` and `<p role="alert">` to
  `aria-describedby`
- **`cn()` for class merging** — `clsx` + `tailwind-merge` means callers can pass a
  `className` prop without worrying about Tailwind class conflicts
- **Accessibility first** — Modals and Drawers use `role="dialog"`, `aria-modal`,
  focus trapping (`useFocusTrap`), scroll locking (`useBodyScrollLock`), and keyboard
  dismiss (`useKeyDown`). Spinners announce themselves via `role="status"`. Errors
  use `role="alert"` for live-region announcements.
- **TypeScript strict** — every prop interface is exported from its component file.
  Types are co-located, not in a separate `types/` folder.

## Hooks

| Hook | Purpose |
|---|---|
| `useKeyDown(key, handler, enabled)` | Attach a keyboard listener while a condition is true |
| `useBodyScrollLock(locked)` | Lock `document.body` scroll while an overlay is open |
| `useFocusTrap(ref, active)` | Cycle Tab focus within a container; restore focus on close |

## Phase 4 Components

### Clients
Components for the client roster and profile screens.

| Component | Stories | Notes |
|---|---|---|
| `SilhouetteAvatar` | Size variants, photo vs placeholder, "Me" badge | Used in cards, profile header, widgets |
| `ClientCard` | Normal, assessment, at-risk, no activity, all states | At-risk gets amber left border |
| `AddClientCard` | Default, in-list context | Pulsing `+` badge — disable by removing `animate-fab-pulse` |

### Dashboard Widgets
Composable tiles ordered by `trainer.widgetProgression`.

| Widget | Stories | Mode |
|---|---|---|
| `AtRiskWidget` | 4 color schemes × 3 tones, empty, single | Trainer only |
| `SelfTrainingWidget` | All CTA labels, with/without goal | Both |
| `ActiveClientsWidget` | 6 clients, empty, one per state | Trainer only |
| `GoalsWidget` | With achievements, empty | Both |
| `RecentSessionsWidget` | Phase 5 placeholder | Both |

| Story | What it shows |
